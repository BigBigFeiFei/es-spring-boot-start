# es-spring-boot-start

es启动器

1.只需要在nacos的yml中配置：


```
elasticsearch:
  address: 10.0.40.13:9200
  username: 
  password: 
```
  
2.然后在项目中引入下依赖:

```
<dependency><br>
     <groupId>org.zlf\</groupId>
     <artifactId>es-spring-boot-start</artifactId>
     <version>1.0-SNAPSHOT</version>
</dependency>
```
